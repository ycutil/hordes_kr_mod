## 4차 적용 사항

`v0.5.0`에서는 번역 축약 없이 실사용 보정을 이어가기 위해 DOM fallback의 긴 문장 치환 제한을 늘렸습니다.

- 기존에는 긴 문장 치환이 보수적으로 제한되어 일부 NPC 대사/튜토리얼 문장이 화면 fallback에서 제외될 수 있었습니다.
- 축약 번역은 하지 않고, `KO_PATCH`의 긴 번역문을 그대로 치환 대상으로 사용합니다.
- 아이템명/스킬명은 사용자가 영어 유지로 결정했으므로 계속 영어로 둡니다.

## 아직 남은 큰 덩어리

공식 한국어 언어팩은 완역 상태가 아니며, 모드가 공식 `ko.json`을 기반으로 쓰는 구조라 아래 영역은 계속 영어가 많이 남습니다.

- `items.*.name`: 장비 이름 대부분 영어 그대로
- `items.book.*.name`: 스킬북/스킬 이름 대부분 영어 그대로
- 아이템명/스킬명: 영어 유지 여부를 결정해야 하는 항목 370개

## 번역 기준 제안

- 게임 내 식별성이 중요한 이름은 일단 영어 유지: `Vanguard`, `Bloodlust`, `Guardstone`, `Headless Landing`, `Gloomfury`, `Obelisk`, `Hellspawn`, `Bone Blessing`
- UI 명령과 안내문은 한국어 우선: `대상`, `장착`, `상인`, `대장장이`, `가속`, `피해`
- 스킬명/아이템명은 한 번에 전부 번역하지 말고, 실제 게임 UI에서 검색성과 채팅 통용성을 확인한 뒤 묶음 단위로 처리
- 용어 통일:
  - Damage: `피해`
  - Haste: `가속`
  - HP: `생명력`
  - Item Find: `아이템 발견`
  - Use/Equip item: `아이템 사용` / `아이템 장착`
